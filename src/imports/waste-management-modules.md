1. Dashboard Overview Module
This is the main page where LGU staff can quickly see the system status.
Features:
•	Total number of smart bins
•	Number of full / near full bins
•	Bins with high Rot Index
•	Active odor neutralization events
•	Recent alerts and notifications
•	Quick statistics (graphs or cards)
Purpose: Gives LGU officials a quick summary of waste conditions across the area.
2. Smart Bin Monitoring Module
Allows LGU personnel to monitor each smart bin in real time.
Features:
•	Bin ID / Location
•	Current weight level
•	Capacity percentage
•	Gas levels (NH₃ and CH₄)
•	Rot Index value
•	Status indicators:
o	Normal
o	Near Full
o	Full
o	High Gas Level
Purpose: Helps LGU know which bins require attention immediately.
3. Map & Location Module
Displays all smart bins on an interactive map.
Features:
•	Map showing bin locations
•	Color indicators:
o	Green – Normal
o	Yellow – Near Full
o	Red – Full / Hazard
•	Clickable bins showing:
o	Weight
o	Gas level
o	Rot Index
o	Last collection time
Purpose: Helps identify which bins should be collected first.
4. Alert & Notification Module
Automatically notifies the LGU when critical conditions occur.
Alerts for:
•	Bin overweight / full
•	High ammonia or methane
•	Rot Index threshold reached
•	Neutralizing mist activated
•	Sensor malfunction
Purpose: Ensures fast response to sanitation risks.
5. Waste Collection Management Module
Used by LGU staff to manage collection operations.
Features:
•	List of bins needing collection
•	Assign collection tasks to workers
•	Collection status:
o	Pending
o	In Progress
o	Completed
•	Record collection time and crew
Purpose: Improves logistics and scheduling of waste collection.
6. Rot Index & Environmental Monitoring Module
Tracks odor and gas data collected from bins.
Features:
•	NH₃ and CH₄ readings
•	Rot Index calculation
•	Historical gas trends
•	Alerts when threshold exceeded
Purpose: Helps monitor health and sanitation risks from decomposing waste.
7. Analytics & Reports Module
Provides data analysis for LGU decision making.
Possible reports:
•	Waste accumulation trends
•	Most frequently filled bins
•	Areas with highest Rot Index
•	Average collection response time
•	Neutralizing system usage
Purpose: Supports data-driven waste management planning.
8. Worker / Personnel Management Module
Manages the sanitation workers using the system.
Features:
•	Add/Edit/Delete worker accounts
•	Assign collection routes
•	Track completed tasks
•	Worker activity logs
Purpose: Organizes LGU personnel responsible for waste collection.
9. Maintenance & System Health Module
Monitors the technical health of IoT devices.
Features:
•	Sensor status
•	Battery / solar power status
•	Connectivity status
•	Maintenance alerts
Purpose: Ensures the IoT hardware operates reliably.
10. User Management & Security Module
Controls access to the system.
Features:
•	Admin accounts
•	LGU staff accounts
•	Worker accounts
•	Role-based permissions
Purpose: Keeps the system secure and organized.
